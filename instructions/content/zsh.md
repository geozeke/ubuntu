# zsh

If you'd like to try a shell other than the default (`bash`), the setup script for the Ubuntu VM installs all the necessary files to run the z-shell with the *Oh My ZSH!* framework for managing the environment.

## Installation

To change your shell, enter this command in a terminal window:

```
chsh
```

Provide your password when prompted, then enter this:

```
/bin/zsh
```

Then *logout* of your current session and log back in for the changes to take effect (no need to reboot).

## Usage

Just use it.
