# gitfiend

Gitfiend is a nice visual Git Client for linux.

## Installation

```Shell
cd ~/Desktop
sudo apt install libcurl4
wget -v https://gitfiend.com/resources/GitFiend_0.22.8_amd64.deb
sudo apt install ./GitFiend_0.22.8_amd64.deb
rm GitFiend_0.22.8_amd64.deb
```

## Usage

Enter:

```Shell
gitfiend &
```

then pin the icon to your favorites.
